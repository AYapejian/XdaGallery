Xda Gallery

General Settings

Number of images to fetch per request: 15

Will continue fetching images until finding this amount, loading one pages worth at a minumum.

Debug Mode: Off

Enabling will turn on Chrome's console logging, should only be enabled to troubleshoot issues.

Advanced Settings

Fetch all images: Off

Enabling will fetch all images in a thread ignoring the number set in the General settings section. NOTE: This can cause problems such as really long fetch times, not displaying images, possibly locking up the tab( just have to close the gallery tab if so), etc... Use at your own risk.
